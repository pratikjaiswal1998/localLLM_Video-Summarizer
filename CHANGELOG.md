# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to Semantic Versioning.

## [1.0.0] - Initial Open Source Release
### Added
- **Core Engine**: Fully offline architecture powered by `Faster-Whisper` for transcription.
- **AI Synthesis**: Seamless local integration with `Ollama` for generating intelligent, context-aware summaries.
- **Presentation Engine**: One-click generation of fully structured, beautifully formatted `16:9` PowerPoint `.pptx` slides.
- **Smart Installer**: Generated automated `Inno Setup` compiler script to bundle necessary dependencies (FFmpeg, Whisper libraries) while intelligently skipping Ollama if present.
- **Marketing Materials**: Included the initial brochure PDF to overview the project.
- **Open Source Governance**: Added `README.md`, `CODE_OF_CONDUCT.md`, `CHANGELOG.md`, and `CONTRIBUTING.md`.
