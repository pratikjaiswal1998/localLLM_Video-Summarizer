# Contributing to AI Video Summarizer

First off, thank you for considering contributing to the AI Video Summarizer! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## How to Contribute

### 1. Reporting Bugs
If you find a bug, please create an issue on GitHub. 
- Ensure the bug was not already reported by searching on GitHub under Issues.
- If you're unable to find an open issue addressing the problem, open a new one. 
- Try to be as detailed as possible in your report (include OS, Ollama version, and stack traces).

### 2. Suggesting Enhancements
Have a cool idea for a new feature? We'd love to hear it!
- Open an Issue and tag it as an enhancement.
- Describe the feature in detail and how it would improve the application.

### 3. Submitting Pull Requests
1. Fork the Project.
2. Create your Feature Branch (`git checkout -b feature/AmazingFeature`).
3. Commit your Changes (`git commit -m 'Add some AmazingFeature'`).
4. Push to the Branch (`git push origin feature/AmazingFeature`).
5. Open a Pull Request.

## Coding Guidelines
- Ensure all Python code follows PEP 8.
- Do not check in massive test video files (use short 5-second mocks if needed).
- Document new methods and classes thoroughly.

Happy coding! 🎓
